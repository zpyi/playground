什么是 Transformer?

Transformer是一种革命性的神经网络架构，于2017年在论文"Attention Is All You Need"中提出。 它完全依赖注意力机制，摒弃了传统的循环和卷积结构。

位置编码

由于Transformer没有循环结构，需要通过位置编码注入序列位置信息。 使用正弦和余弦函数生成位置向量。

PE(pos, 2i) = sin(pos / 100002i/d)
PE(pos, 2i+1) = cos(pos / 100002i/d)

自注意力机制

自注意力让模型理解词与词之间的关系。每个词可以"注意"到序列中的其他所有词。

Attention(Q, K, V) = softmax( QKT / √dk) V

Query (Q)

查询向量：我想要找什么

Key (K)

键向量：我是什么

Value (V)

值向量：我的内容

多头注意力

多头注意力让模型同时关注不同位置的不同表示子空间，捕捉多种语义关系。

MultiHead(Q,K,V) = Concat(head1, ..., headh)WO
Head 1 - 语法关系
Head 2 - 语义关系
Head 3 - 指代关系
Head 4 - 位置关系
